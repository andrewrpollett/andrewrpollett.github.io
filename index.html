Electrohydrodynamic oil-in-oil emulsion simulations

Using the Basilisk fluid flow solver, simulations of droplets in an oil-in-oil emulsion subjected to an external, uniform electric field are developed. Using these simulations, fundamental limitations involved with many common simulation techniques of this problem can be investigated.
