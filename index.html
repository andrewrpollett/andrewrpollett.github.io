Research Interests

Distributed Cloud Storage Systems

Electrohydrodynamic oil-in-oil emulsion simulations

Using the Basilisk fluid flow solver, simulations of droplets in an oil-in-oil emulsion subjected to an external, uniform electric field are developed. Using these simulations, the limitations involved with many common simulation assumptions of this problem, such as assuming a 2D domain, can be investigated. This can be used to quantify the discrepancy between simulations and corresponding real-world experiments.
